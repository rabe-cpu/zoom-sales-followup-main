# 前田 季輝さん

## 商談内で見えた特徴

- `そしたら`（口癖。話の切り替え・次の案内で多用）
- `〜していただいてもいいですか？`（操作案内・依頼で非常に頻出。丁寧で押しつけない）
- `〜していただければと思います`（依頼・案内の定番着地）
- `なんで`（＝だから、の接続）／`やっぱり`／`実際に`／`一応`／`ちなみに`
- `〜という形になります` / `〜という形です`（説明の着地）
- `〜なイメージです`（収益などの説明の締め）
- `大丈夫そうですか` / `ご理解いただけてますか`（小まめな理解確認）
- `ご安心ください` / `承知いたしました` / `少々お待ちください` / `失礼いたします`（丁寧な進行）
- `ありがとうございます`（相槌・お礼として頻出。`めちゃくちゃいい質問です` と顧客を立てる）

## 一人称・距離感

- 一人称: 僕 / 私（`僕の方から`、`私からご連絡差し上げます`、`私たちの方から`）、弊社
- 顧客への呼び方: `井上さん` / `井上様`（さん・様を併用）
- 不慣れな顧客にも辛抱強く、一歩ずつ確認しながら進めるスタイル

## メール反映の定型

- `そしたら、〜していただければと思います`
- `〜していただいてもよろしいですか`（依頼の柔らかい着地）
- `また私からご連絡差し上げます`
- `資料をお送りしますので、ゆっくりご覧いただければと思います`
- `ご不明点やご質問があれば、おっしゃっていただければと思います`
- 締め: `お時間いただきましてありがとうございました`

## 余白表現の出現

- `かと`: あり（`わかりやすいかと思います`、`なかなか出せないかと思います`、`お話しさせていただいたかと思います`）→ 1通0〜1回までOK
- `かなと` / `かな`: あり（`落ち着いてくるかなという`、`なってくるかなってところ`）→ 0〜1回までOK
- `できればと`: あり（`ご紹介できればと思います`、`お話しさせていただければと思います`）→ 着地として自然に使用可。ただし依頼文を曖昧に止めない
- `見ていただけると`: `見ていただければと思います` の形で頻出 → `見ていただけますと助かります` 等の着地ならOK

## 禁止候補

- `伝わってまいりました`（前田さんの発話に確認なし）
- 成果保証・断定（商談では `必ず比例` `やった分だけ増える` と語るが、メールでは数字を保証にしない）
- 過度な美文
- 顧客の不安の断定・煽り（井上さまは初期利益や手続きに不安を口にしており、丁寧に受け止める）
