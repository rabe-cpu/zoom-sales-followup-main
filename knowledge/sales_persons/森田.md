# 森田 優弥さん

## 商談内で見えた特徴

- `結論から言うと`（口癖。複数回。質問への切り出しで多用）
- `要は`（要点を言い換えるときの接続）
- `もっと言うと`（補足を足すときの接続）
- `単純に`（仕組みをかみ砕くときの接続）
- `そもそも`
- `逆に`（`逆に何かご不明点とか`）
- `〜という形です` / `〜という形`（締めの言い回し）
- `〜なイメージです` / `イメージとしては`（説明の着地）
- `ありがとうございます`（相槌・お礼として非常に頻出）
- `〜で大丈夫です`（不安をほどくときの定番。`考えなくて大丈夫`、`集客しなくても大丈夫`、`毎日やらなくて大丈夫`）

## 一人称・距離感

- 一人称: 僕 / 僕ら（`僕の方で`、`僕らのシステム`、`僕の中では`）、改まると 弊社 / 私
- 顧客への呼び方: `〇〇さん`（商談中は基本さん付け）

## メール反映の定型

- `結論から言うと、〇〇です`（要点を先に出す）
- `要は〇〇という形です`
- `単純に〇〇していくだけです`（仕組みをかみ砕く）
- `〇〇で大丈夫です`（不安への受け止め）
- `〜なイメージです`（説明の締め）
- 締め: `よろしくお願いします`

## 余白表現の出現

- `かなと`: あり（`以上かなと思います`）→ 1通あたり0〜1回までOK
- `かと`: 単独での明確な出現は確認できず → 使わない
- `できればと`: 出現なし → ネクストアクションは「いただけますと助かります」「ご連絡いただけますでしょうか」で着地させる
- `見ていただけると`: 出現なし → 使わない

## 禁止候補

- `伝わってまいりました`（森田さんの発話に確認なし）
- 成果保証・断定（商談では `絶対出せる` `ほとんど達成している` と語る場面があるが、本人も `絶対というか` と言い直している。メールでは `〜という方が多いです` 程度に留め、保証表現にしない）
- 過度な美文
- 顧客感情の断定
