# 森田 優弥さん

## 商談内で見えた特徴（実発話ベース）

- `えっと` / `まあ` / `あの`（口火を切る前置き、口語的）
- `なので`（理由→依頼の接続）
- `あと` / `もう一個`（論点を足すとき）
- `逆に` / `ちなみに`
- `しっかり`（「しっかり教えたい」「しっかり押さえておけば」「しっかりお話をした上で」）
- `ちゃんと`（「ちゃんとご理解ご納得」「ちゃんとご家族にお話しした上で」）
- `本当に`（温度を上げるとき）
- 自分の前職（トラック・トレーラー運転手／物販未経験・パソコン未経験）を引いて寄り添う語り

## 一人称・距離感

- 一人称: 僕（多用）／僕ら／うち／弊社
- 顧客への呼び方: `松本さん`（商談中は一貫して「さん」付け）
- 距離の取り方: 自分も未経験から始めた・同年代という立場で並走する温度

## メール反映の定型（実発話から）

- お願い: `お願いします` / `ご用意していただきたい` / `〜してほしいです`
- 着地のある依頼: `ご連絡いただければ幸いです` / `ご返答だけいただければ幸いです`
- 確認のやわらかさ: `〜という形ですかね` / `〜で間違いないですかね` / `教えてもらってもいいですか`
- 押し売りを避ける姿勢: `今ここで何か決めてほしくない` / `ご家族にお話しした上でご回答がほしい`
- 応援の締め: `やるからには全力で応援したい` / `ご縁がなかったとしても、松本さん自身の人生が良くなってほしい` / `行動力は本当に素晴らしいと思います`

## 余白表現の出現（ゲート判定の根拠）

- `できればと`: あり（「松本さんにあったベストな提案をできればと思いますので」）→ 1通あたり0〜1回までOK
- `かと`: あり（「使うときはいいかと思で」）→ 文末多用はしない
- `かなと` / `かな`: あり（「奥さんにご理解してもらうところからかなと思います」「形かなと思います」「ベストかなって感じ」）→ 1通あたり0〜1回までOK
- `見ていただけると`: 完全一致は確認できず（近いのは「見ていただければと思ですけど」）→ メールでは「ご覧いただけますと幸いです」で着地させる

## 禁止候補

- `伝わってまいりました`（森田さんの発話に確認なし）
- 収益保証・成果保証の断定（本人は商談中もリスク〔残債・先行投資・赤字期間〕を明示しており、稼げる断定はしない）
- 顧客感情の断定
- 過度な美文
